# Vizuelne reference

Ovde dodajte odobrene grafike, kratke video-reference ili snimke ekrana čiji se kvalitet i pristup mogu koristiti kao inspiracija za buduće objave.

Agent ih koristi isključivo za art direkciju, kompoziciju, hijerarhiju i obradu, ali ne kopira doslovno raspored, tekst, brend elemente ili prepoznatljiva kreativna rešenja drugih brendova.

Reference mogu pripadati potpuno drugim nišama. Ne preuzimaju se njihova tema, proizvod, zdravstvene tvrdnje, copy, CTA, publika, cena, rokovi ni brend kontekst. Za AU Šeki-Tiliju i dalje važe samo potvrđene činjenice, njen brend vodič i bezbednosna pravila.
