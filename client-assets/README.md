# Lokalni materijali klijenta

Ovaj folder je namenjen privatnim, izvornim fotografijama i drugim materijalima koje dostavlja AU Šeki-Tilija. Sadržaj foldera se ne objavljuje na GitHub-u.

## Fotografije apoteka

Za svaku apoteku napravi poseban folder koji koristi isti stabilni ID kao `brand/brand-config.json`:

```text
client-assets/
  locations/
    petrovac-na-mlavi-centar/
      eksterijer-01.jpg
      enterijer-01.jpg
    veliko-gradiste-glavna/
      eksterijer-01.jpg
```

Ne menjaj originalna imena fajlova ako bi to otežalo povezivanje sa materijalom klijenta. Ako je potrebno, dodaj kratku `README.md` belešku u folder lokacije.

## Šta još treba prikupiti

- tačnu adresu, telefon i radno vreme svake apoteke;
- potvrdu eventualnih odstupanja u radnom vremenu;
- fotografije apoteka za koje postoji dozvola javne upotrebe;
- fotografije proizvoda i zvanične informacije za buduće akcije;
- potvrđene cene, popuste, rokove akcija i lokacije na kojima akcija važi.

Agent koristi samo potvrđene podatke. Ako nešto nedostaje, označava to u `review.md` umesto da pretpostavlja.
